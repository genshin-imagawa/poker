# POKER

## 概要

ポーカーの役判定アプリケーション<br>
柔軟な思考でロジックを組み立てることを目的に作成

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
